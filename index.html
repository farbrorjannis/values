🎨 Value Tool – Find the right tones

Simplify your reference image into few values – perfect for watercolor sketches

💡 Tip: 3 values are best for finding big shapes. 5-8 values provide more detail.

📖 About Values

Values are the levels of lightness in an image. By simplifying an image into just a few values, you can see the essential shapes and contrasts more clearly. This is a perfect foundation for watercolor sketches where you want to plan light and dark areas.
