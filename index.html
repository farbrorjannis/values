🎨 Value Tool – Find the Right Tones

Simplify your reference image into core values – perfect for watercolor sketches

💡 Tip: 3 values are best for finding big shapes. 5-8 values provide more detail.

📖 About Values

Values represent the lightness or darkness of a color. By simplifying an image, you can see the essential shapes and contrasts more clearly.
